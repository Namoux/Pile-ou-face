Présentation personnelle : Je m'appelle Nam, et je suis actuellement en formation DWWM.

Synopsis : Je vais vous présenter mon projet qui consiste en un jeu simple et amusant : "Pile ou Face". 
L'objectif de ce projet était de créer une application interactive où les utilisateurs peuvent deviner 
le résultat d'un lancer de pièce.

Cas d'utilisation : Avec mon application "Pile ou Face", l'utilisateur peut :

    Choisir entre "Pile" et "Face".

    Voir le résultat du lancer de pièce.

    Suivre ses réussites et ses échecs.

    Quitter le jeu à tout moment.

Stack technique : Pour réaliser ce projet, j'ai utilisé les technologies suivantes :

    Langage de programmation : C.

    Compilateur : GCC.

    Système d'exploitation : Debian 12.

    Environnement de développement : Visual Studio Code.

Code : Le projet se compose principalement de deux parties :

    La fonction saisie() :

        Elle gère les entrées de l'utilisateur, vérifie les choix et traite les saisies 
        en majuscules et minuscules.

    La fonction main() :

        Elle lance le jeu, gère les lancers de pièce et maintient le décompte des réussites et des échecs.

Difficultées : Pendant le développement de ce projet, j'ai rencontré plusieurs difficultés, notamment :

    La gestion des entrées de l'utilisateur, en particulier pour traiter les majuscules et les minuscules.

    L'utilisation de la fonction atoi et la vérification correcte des entrées numériques et textuelles.

    L'implémentation d'un générateur de nombres aléatoires pour simuler le lancer de pièce.

Améliorations futures : Pour les futures versions de ce projet, je prévois les améliorations suivantes :

    Utiliser plus de fonctions pour eviter de surcharger le main.

Au revoir : Voilà, c'était la présentation de mon projet "Pile ou Face".
